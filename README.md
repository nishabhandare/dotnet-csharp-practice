# C# Data Types and GetType() Demo

## Overview

This project demonstrates the use of various built-in data types in C# and how to identify their data types using the `GetType()` method.

## Data Types Used

* char
* int
* short
* long
* uint
* ushort
* ulong
* float
* double
* decimal

## Features

* Declares variables of different C# data types.
* Assigns sample values to each variable.
* Displays the value and its corresponding data type.
* Demonstrates the use of the `GetType()` method.

## Technologies Used

* C#
* .NET Console Application

## Learning Outcomes

* Understanding C# data types.
* Difference between signed and unsigned numeric types.
* Using the `GetType()` method.
* Basic console output operations.

## Sample Output

The program prints each variable value along with its data type in the console window.



# Nullable Type Example in C#

A simple C# program demonstrating the use of Nullable Types (`Nullable<T>`).

## Description

This program shows how nullable data types can store `null` values and how the `GetValueOrDefault()` method returns the default value of the underlying data type when the nullable variable contains `null`.

## Features

* Demonstrates Nullable Types in C#
* Uses `Nullable<int>`, `Nullable<float>`, and `Nullable<bool>`
* Uses `GetValueOrDefault()` method
* Displays default values of different data types

## Technologies Used

* C#
* .NET Console Application

## Sample Output

```text
Nullable Type Example
This program checks the default values of data types using nullable types.

Default value of int is: 0
Default value of float is: 0
Default value of bool is: False
```

## Learning Outcome

* Understanding Nullable Types (`Nullable<T>` or `T?`)
* Handling null values safely
* Using `GetValueOrDefault()` to retrieve default values
